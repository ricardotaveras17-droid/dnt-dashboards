WHAT TO PASTE
PASTE CSV

Option A — Raw QuickBooks CSV (easiest)

In QBO, run Reports → Transaction List by Date (or Transaction Detail by Account), filter to your 11 marketing/labor/overhead accounts for the period, hit Export → CSV. Paste the whole thing. Dashboard detects QBO columns (Date, Account, Name, Amount) and auto-maps each account to its Parent + Category. Accounts it can’t map are skipped with a warning.

Option B — Preprocessed Transactions CSV

Header must include: Date, Parent, Category, Vendor, Amount. The output of parse_pdfs.py — use this if you already ran the Python pipeline.

Option C — Sift Attribution CSV (adds CPL/ROI)

Header must include: Month, Channel, Spend, Leads, CPL, Contracts, Revenue, ROI. Output of sift_attribution.py. Layered on top of A or B.
